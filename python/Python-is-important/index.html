    📠Ascii码-Unicode编码-UTF-8编码

    pta和课本上有相关题目,因此做简单的概念区分

    • "Ascii码":用二进制编码来表示非数值的文字与符号(数字化),如今普遍采用的就是ASCII(美国信息交换标准代码),有7位二进制与八位二进制两种版本(国际通用7位,有\(2^7 = 128\)个元素),与 Unicode 编码和 UTF-8 编码兼容

      字符 A:65,a:97,Z:90,z:122

    • "Unicode编码":包含了世界上几乎所有的字符，包括各种语言的文字、符号、表情符号等。它是目前使用最广泛的字符编码方案，可以满足不同语言和文化之间的交流需求。Python3中的字符串是Unicode字符串而不是字节数组.

    • "UTF-8":是 Unicode 编码的一种实现方式,使用1到4个字节来表示不同的字符,特点是对不同范围的字符使用不同长度的编码(不等长).对于0x00-0x7f之间的字符,UTF8与ascii编码完全一致.

    ✨F-string (important)

    F-string是一种格式化字符串常量，它允许我们在字符串中插入变量或表达式，并以指定的格式输出。

    在格式化的字符串中，使用大括号 {} 来表示需要被替换的字段，而不在大括号 {} 内的内容将会被原样输出。如果想要输出大括号 {} 字符本身，可以使用双层大括号 {{}} 将其包裹起来。通过在大括号 {} 中直接填入要替换的内容，我们可以轻松地生成格式化后的字符串。
